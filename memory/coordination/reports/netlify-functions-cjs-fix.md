## Decisions

- `.cjs` rename over `netlify/functions/package.json {"type":"commonjs"}`:
  the deploy bundler flattens the function file to `/var/task/` next to the
  root package.json (proven by the live stack trace), so a nested package.json
  is not guaranteed to govern parsing there; a file extension always is.
- Discovered mid-task: `netlify/__tests__/*.test.js` read function source by
  filesystem path (missed in the original card — test count dropped 268→235).
  Card amended by coordinator; paths updated; count restored to 268.
